Experiment 4.2 - Space Watching

Description

This is the second hashed random isometric tile project. The eyes follow your cursor and go "disco mode" when clicked. The backround was originally supposed to represent a bowling alley floor but ended up representing a starry sky so we just kept it like that.

Technical

The position of the eyes and the stars on the ground are all randomly generated based off of a hashed phrase used as a seed for the random number generator.

Reflection

I helped with the technical aspects of porting the project from glitch and to a local server as well as helping Sean bug fix.
