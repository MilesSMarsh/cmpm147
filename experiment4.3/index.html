Experiment 4.3 - We Could've Made This In Phaser

Description

Part 3 of messing with an isometric tile seeded random generator. In this one the world is supposed to represent a mad max aesthetic with mostly dirt and an occasional water tile

Technical

Using a phrase as the key to a hash function we then take that hashed phrase and use it to seed a random nuber generator to create the background tiling. The randomness is not so notable here as the tiles move by realy fast and don't have any defining features to make them worth hashing. The car that chases you does change shape and is shaped based on the seeded RNG so it will be the same if you give the same world key.
